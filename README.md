# EPS C20 Practice Console

A local browser practice interface for the EPS C20 question bank.

## Open it

Open `index.html` in a browser, or run a small local server from this folder:

```bash
python3 -m http.server 8765
```

Then visit:

```text
http://localhost:8765/index.html
```

## Features

- 282 questions from the workbook
- Filter by question set, lecture/source, and topic
- Shuffle mode
- Exam mode
- Answer checking with explanations
- Missed-question review
- Browser-saved session progress
